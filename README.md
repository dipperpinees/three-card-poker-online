## Install packages
#### `npm install && npm install --cwd client` 
or
####  `yarn install && yarn install --cwd client`

## How to start
### `npm start` or `yarn start`
Open [http://localhost:3000](http://localhost:3000) to view it in the browser (back-end running on port 8021).
